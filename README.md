# CSES Problem Set — Python Solutions

Personal solutions to competitive programming problems from the **[CSES Problem Set](https://cses.fi/problemset/list/)** — a curated collection of algorithmic challenges covering classical CS topics.

---

## Problem Categories

The CSES set covers:
- **Introductory Problems** — basic loops, math, strings
- **Sorting and Searching** — binary search, two pointers, sorting tricks
- **Dynamic Programming** — knapsack, LCS, path problems
- **Graph Algorithms** — BFS, DFS, shortest paths, topological sort
- **Tree Algorithms** — LCA, diameter, tree DP
- **Mathematics** — modular arithmetic, primes, combinatorics
- **String Algorithms** — Z-algorithm, suffix arrays, Aho-Corasick
- **Geometry** — convex hull, line intersection

---

## Usage

Each solution is a standalone Python file. Run any with:

```bash
python solution.py
```

Input is typically read from stdin — paste the problem's sample input or pipe from a file:

```bash
python solution.py < input.txt
```
